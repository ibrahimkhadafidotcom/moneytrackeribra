MONEY TRACKER v7.3 — AUTO BUDGET DARI ACTUAL INCOME

Perubahan utama:

1. Budget Tabungan dihitung otomatis sebesar 20% dari actual income periode aktif.
2. Budget Sedekah dihitung otomatis sebesar 2,5% dari actual income periode aktif.
3. Actual income khusus berasal dari Gaji Bulanan, Freelance, Bonus, dan Penjualan.
4. Nilai otomatis diperbarui saat transaksi ditambah, diedit, dihapus, periode aktif diganti, atau data cloud diperbarui.
5. Budget otomatis ditandai pada Dashboard dan halaman Budgeting serta tidak dapat diubah manual.
6. Master Tabungan dan Sedekah dibuat otomatis jika belum tersedia.
7. Cache PWA dinaikkan ke v7.3 agar pembaruan langsung terbaca setelah deployment.
