Masuk / Buat Akun

Pakai email dan password yang sama di MacBook dan iPhone. Password minimal 6 karakter.

Firebase belum dihubungkan. Ikuti tutorial di file TUTORIAL_BAHASA_BAYI.md.

Tombol Sinkronisasi

Pertama kali: buka device yang datanya paling lengkap, login, lalu tekan Kirim Data Device Ini ke Cloud.

Setelah setup pertama selesai, sinkron berjalan otomatis. Saat internet mati, data tetap tersimpan di device dan akan dikirim lagi saat internet hidup.

Cara Pakai Super Gampang

1
MacBook dulu.
Pakai device yang datanya paling lengkap.
2
Login atau buat akun.
Pakai email dan password pribadi.
3
Tekan “Kirim Data Device Ini ke Cloud”.
Data MacBook naik ke awan Firebase.
4
Buka app di iPhone.
Login pakai akun yang sama. Data akan turun sendiri.
5
Selesai.
Input di iPhone akan muncul di MacBook, dan sebaliknya.

Backup & Export

Backup JSON menyimpan semua data: transaksi, kategori, rekening, periode, budget plan, dan Money Plan 50/30/20.

Import Backup / Laporan Sheets

File laporan dengan mode merge akan ditambahkan tanpa menghapus data yang sudah ada. Transaksi yang sama akan dilewati agar tidak dobel. Backup JSON biasa tetap dapat mengganti seluruh data setelah konfirmasi.

Cara Install Jadi App

  1. Buka app lewat Chrome/Safari/Edge.
  2. Pilih Add to Home Screen / Install App.
  3. Untuk offline stabil, buka lewat local server: python3 -m http.server 8000.
