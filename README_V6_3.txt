MONEY TRACKER v6.3 — DEBT DI DALAM NEEDS 50%

LOGIKA FINAL
============

Dana Expected =
Expected Gaji Bulanan
+ Expected Bonus
+ Expected Freelance

Pembagian:
- Needs + Debt = 50%
- Wants = 30%
- Saving & Investasi = 20%

Debt Motor:
- Default Rp1.730.000 per periode
- Debt mengambil bagian dari alokasi Needs 50%
- Sisa Needs = 50% Dana Expected - Debt

Contoh:
Dana Expected Rp8.000.000
- Needs + Debt 50% = Rp4.000.000
- Debt Motor = Rp1.730.000
- Needs Lain = Rp2.270.000
- Wants = Rp2.400.000
- Saving = Rp1.600.000

BILA DEBT LEBIH BESAR DARI 50%
==============================
Aplikasi memberi peringatan bahwa alokasi tidak sehat.
Needs Lain menjadi Rp0 dan Wants sebaiknya dikurangi.

CARA MENGUBAH TARGET DEBT
=========================
Default otomatis Rp1.730.000.
Untuk mengganti:
1. Buka Budgeting.
2. Group: Debt.
3. Category: Cicilan Motor.
4. Isi nominal Expected baru.
5. Simpan.

ACTUAL DEBT
===========

Transaksi Hutang atau pinjaman masuk tidak dianggap pembayaran cicilan.
